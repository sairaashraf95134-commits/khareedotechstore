

# 🛍️ Welcome to Khareedo Tech Store

A modern, responsive e-commerce web application for a tech store, built with [Lovable](https://lovable.dev/). This project showcases a clean user interface for browsing and purchasing electronics and gadgets.

**Live Demo:** [khareedotechstore.lovable.app](https://khareedotechstore.lovable.app/)

## ✨ Features

- **Product Catalog:** Browse a curated list of tech products with images, descriptions, and pricing.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Modern UI Components:** Clean card layouts, navigation bars, and product detail views generated by Lovable AI.
- **Shopping Cart Simulation:** Add items to cart and manage quantities (state management via React hooks).
- **Seamless Lovable Sync:** This repository is connected to Lovable for two-way sync, allowing for rapid AI-assisted development.

## 🛠️ Tech Stack

This project was generated using the Lovable platform, which utilizes a modern front-end stack:

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui (Radix UI primitives)
- **State Management:** React Context API / TanStack Query
- **Routing:** React Router DOM
